# Landin Page Example

Landing page moderna servida con Node.js. Incluye una estructura sencilla, estilos responsivos y JavaScript ligero para interacciones de navegación y revelado de contenido.

## Requisitos

- Node.js 18 o superior
- npm

## Instalacion

No hay dependencias externas. Puedes ejecutar el proyecto directamente con Node.js.

## Ejecucion

```bash
npm start
```

Abre `http://localhost:3000` en el navegador.

## Desarrollo

```bash
npm run dev
```
